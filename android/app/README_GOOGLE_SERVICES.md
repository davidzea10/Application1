# Fichier google-services.json

**IMPORTANT** : Ce fichier doit être téléchargé depuis Firebase Console.

1. Aller sur [Firebase Console](https://console.firebase.google.com/)
2. Sélectionner votre projet
3. Aller dans Paramètres du projet > Vos applications
4. Télécharger le fichier `google-services.json` pour Android
5. **Placer ce fichier ici** : `android/app/google-services.json`

Le fichier doit être nommé exactement : `google-services.json`
